# React Course Project

This project was created as part of the Udemy course:  
**React – The Complete Guide 2025 (incl. Next.js, Redux)** by Maximilian Schwarzmüller.

## Getting Started locally

1. Install dependencies:

```bash
cd backend
npm install
cd frontend
npm install
```

2. Run App
```bash
cd backend
npm start
cd frontend
npm start
```
